## 소개

프론트엔드부터 백엔드, 서버 구축, 네트워크 보안까지 다루는 풀스택 개발자 Kalpha_Dev의 포트폴리오 웹사이트입니다. 최신 웹 기술과 디자인 트렌드를 반영하여 제작되었으며, 반응형 디자인과 접근성을 고려하여 개발되었습니다.

## 주요 기능

- **반응형 디자인**: 모바일, 태블릿, 데스크톱 모든 디바이스 지원
- **부드러운 애니메이션**: 스크롤 기반 요소 표시 및 로딩 화면 애니메이션
- **타이핑 효과**: 동적 텍스트 타이핑 애니메이션
- **접근성 최적화**: ARIA 레이블, 키보드 네비게이션, 스킵 링크 지원
- **SEO 최적화**: 메타 태그, 구조화된 데이터(JSON-LD), sitemap.xml
- **성능 최적화**: 이미지 lazy loading, 최적화된 폰트 로딩

## 프로젝트 구조

```

├── index.html          # 메인 HTML 파일
```

## 로컬 개발

### 설치 및 실행

```bash
# 저장소 클론
git clone https://github.com/gguatit/portofolio-main.git

# 디렉토리 이동
cd portofolio-main

# 로컬 서버 실행 (Python 사용)
python -m http.server 8000

# 또는 (Node.js 사용)
npx http-server -p 8000
```

브라우저에서 `http://localhost:8000` 접속
